OS X El Capitan

  • life

OS X El Capitan 升级之后字体更加好看了，mission control 也是更加的好用，至于分屏，我还是强烈推荐 Hyperdock，还提供预览效果，比原生不知道强到哪里去了。

El Capitan 增加了一个叫做 System Integrity Protection 的保护机制，我只能说很烦, /usr /System 以及 /bin 无法被写入，很多软件设定需要改，这里记录一下改动，说不定哪一天还要再改回去。。。

首先最重要的是关闭 sip：
进入 recovery mode（开机时死按住 command + R), 然后打开命令行输入:
csrutil disable

需要重装或设定的软件名单：

  1. homebrew: 把 Xcode 升到7.0.1以上,然后 terminal 输入 sudo chown -R $(whoami):admin /usr/local
  2. iStat Menus: 5.11以上 OK。
  3. clean my Mac: 更新到最新可用。
  4. flashlight: 很可惜，但是作者表示应该无法支持。参见, 我还蛮欣赏作者的态度如下：

    Apple’s new “rootless” security system blocks code injection into system processes. Technically you can disable rootless – but I’ll have to agree with the TotalFinder devs that it doesn’t make sense to ask ordinary users to do this. And these security restrictions will only get tighter in the future – it doesn’t make sense to build a product around circumventing them.

  5. MacTex失效，卸载了装最新 MacTex2015.
