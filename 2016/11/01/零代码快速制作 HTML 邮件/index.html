零代码快速制作 HTML 邮件

什么是 HTML 邮件

每个人的邮箱中应该都少不了一些广告订阅邮件，这样的邮件，看起来就和一张网页一样。很多人的一天工作，就是从打开这些邮件瞟一眼，然后把他们纷纷扫入垃圾箱开始。

有一天，我想和实验室同事们分享一个英语写作润色软件 Grammarly ，于是也想试着做一个这样的邮件，发送之后收到好评。 下面给出一个我发送的 HTML 邮件的 Demo 。

从这个 Demo 中可以看到简单的介绍后，我列出了两种使用场景以供挑选，Mac 用户可以选择在线服务，Windows 用户则可以安装 Word 插件。 显眼的红色 button 则能够迅速的提供链接访问或者下载地址，非常的清晰明了。仔细思索一下，HTML 邮件还是具有自己的突出特色：

  • 更加美观的图文混排。
  • 明显的按钮 (Button) 显示，刺激点击。
  • 省去了附件的添加，大多数人是不愿意下载附件的。

综于以上优点，HTML 邮件特别适合用于营销，教程，贺卡等应用场景，在工作中也具有一定的实用价值，值得我分享一下自己的使用经验。

制作 HTML 邮件

如果你从零开始制作 HTML 邮件， 你会发现它是一件有挑战的工作。由于 Email 对于 HTML 没有统一的标准，我们平时编写的 HTML 文件， 最后呈现效果很大程度上取决于你的邮件服务商以及客户端。为了达到统一满意的效果， HTML 邮件最好使用 XHTML 版本，利用嵌套的表格 (Table) 进行整体的布局，并把所有 CSS 写成行内 (Inline) 样式。

当然，今天的文章不会牵扯到任何代码，因为这些老旧的网页制作方法早已被时代抛弃，实际上已经失去了学习的价值。我们完全可以通过网络提供的服务来快速制作自己的 HTML 模板，下面推荐的网站 Mailchimp 就可以帮我们快速完成制作，并贴心地提供了源代码导出。

首先我们先注册一个账号，注册时的公司名可以随意无需讲究，这个网站的密码强度要求还挺高的，不仅要求大小写数字，还要求包括特殊字符，大家注册时根据提示多加小心。注册完之后，即可开始创建属于自己的模板。

创建模板时，网站贴心的提供了三种模式，「Basic」，「Themes」，「Code your own」。

「Basic」 模式是从排版布局出发，有点像幻灯片母版的感觉，之后的所有内容都要自己添加。「Themes」则是给提供一个参考的主题，你可以从中找到自己的灵感，当然，里面的文字图片你可以进行二次修改，快速达到自己的需求。
「Code your own」可以导入自己的代码，是进阶使用者大展身手的地方。

接下来我们要谈的是它的编辑模式，所有的操作可以用「Drag and Drop」 来形容，网站贴心的提供了常见的小部件，我们所要做的仅仅是把所需部件放在想要的地方。

放完所需部件后，我们可以对内部文字的格式进行进一步的调整：字体大小，居中对齐，等等。内置的图形化编辑器非常的易用，很多细节都可以进行微调，整个的体验可以说就像做一个 PPT 一样，熟练之后非常的行云流水。

完成之后，我们最终选择导出为 HTML 。一个贴心的细节是我们上传的图片在 HTML 文件中显示为「绝对 URL 」，这意味着我们 HTML 邮件中的图片可以被他人直接浏览，而不需要自己再上传图床做链接，真的是足够贴心呢。

最后，Mailchimp 导出的 HTML 还不够完美，如上文提及，我们最好把所有 CSS 全部写成行内 (Inline) 样式，有很多热心的程序猿们提供了不同脚本，为了方便，我们采用在线网站 Magic CSS Inliner Tool 进行转换：

最后的结果另存为新 HTML 文件，到此为止，一份可用的 HTML 邮件已经制作完毕，完事具备，只欠发送。

发送 HTML 邮件

1. 通过 QQ 邮箱发送

作为常年受歧视的国内邮箱代表，用QQ 邮箱发送 HTML 邮件却意外的好用， 编辑栏中自带有 HTML 编辑模式，在 HTML 模式下清空已有内容，把代码贴入，再次转回可视化编辑模式即可。

2. 通过 Gmail 发送

Gmail 自身并没有提供 HTML 编辑模式，但可以我们可以通过 chrome 浏览器内置的调试台进行编辑。为了方便编辑，建议在空白发件箱中先打一段准备替换的文字，比如「参考文本」。

第一步：选中「参考文本」并右键单击选择「检查」

第二步：在对应的部分右键单击选择「Edit as HTML」

第三步： 找到我们的「参考文本」，替换为我们之前得到的 HTML 代码。点击空白处保存返回即可。
